전체 0개
📱 안성 남부 카페 구역 · 사용 안내
🔐 로그인 방법

등록된 이름과 비밀번호를 입력해 로그인합니다.

로그인 후 본인 이름으로 방문 표시와 보고서가 저장됩니다.

비밀번호는 우측 상단 🔑 비번 변경 버튼으로 언제든 바꿀 수 있습니다.

📊 담당자 카드 필터

화면 상단에 담당자별 카드(하명수 / 임형규 / 오주명 / 프랜차이즈)가 표시됩니다.

카드를 클릭하면 해당 담당자의 카페만 필터링됩니다. 카드에 ✓ 선택 표시가 나타납니다.

같은 카드를 다시 클릭하거나 상단 전체 탭을 누르면 전체 목록으로 돌아옵니다.

각 카드에는 담당 카페 수와 보고서 작성 건수가 함께 표시됩니다.

🔍 검색 & 필터

상단 검색창에 상호명·주소·동 이름을 입력하면 실시간으로 필터링됩니다.

카드 필터 아래 동/면 버튼을 누르면 해당 지역 카페만 표시됩니다.

정렬 버튼으로 번호순 / 카페명순 / 담당자순으로 정렬할 수 있습니다.

🚶 방문 참여 기능

카페 행의 방문 참여 버튼을 누르면 방문자를 지정할 수 있습니다.

지정된 방문자 이름이 목록에 배지로 표시되고, 통계 카드가 업데이트됩니다.

방문을 마치면 🚶 방문 중 버튼을 다시 눌러 해제하세요.

상단 🚶 내 방문 현황 탭 또는 방문 현황 카드를 클릭하면 내가 방문 중인 카페만 모아볼 수 있습니다.

🔄 구역 변경 / 폐업 처리

카페 행의 🔄 구역 버튼을 누르면 담당자 변경 또는 폐업 처리를 할 수 있습니다.

  • 담당자 변경 – 잘못 배정된 구역을 다른 담당자로 이동합니다.
  • 🗑 폐업 삭제 – 폐업한 카페를 목록에서 제외합니다.

변경 사항은 모든 사용자에게 실시간으로 반영됩니다.

📝 보고서 작성 순서
  1. 카페 행에서 📋 작성 버튼을 누릅니다.
  2. 바로 아래 펼쳐지는 폼에 내용을 작성합니다.
  3. 💾 저장 버튼을 눌러야 최종 저장됩니다.
  4. 저장된 카페는 📝 수정 버튼으로 표시됩니다.

작성 항목: 기본정보 · 업종정보 · 영업시간 · 구역조사 · 실전준비 · 방문기록 · 후속조치 · 보완사항

🕐 영업 시간 입력 방법

기본 시간에 대부분 요일의 공통 영업 시간을 입력합니다. (예: 10:00 ~ 21:00)

요일 버튼(월~일)을 클릭해 영업일/휴무일을 전환합니다.

  • ● 파랑 = 영업일  |  ● 회색 = 휴무일

특정 요일의 시간 칸을 비워두면 기본 시간이 자동 적용됩니다.

📅 방문 기록 작성 방법

보고서의 방문 기록 표에서 방문일·대화 요약·방문자·결과를 입력합니다.

📅 달력 클릭을 누르면 달력이 열려 날짜를 선택할 수 있습니다.

+ 행 추가 버튼으로 방문 기록을 늘릴 수 있습니다.

🗺 지도 기능

상단 🗺 지도 탭을 누르면 전체 카페 위치가 지도에 표시됩니다.

  • 마커 클릭 – 카페 정보와 네이버 지도 → 링크가 팝업으로 표시됩니다.
  • 📍 내 위치 – 현재 내 GPS 위치를 지도에 파란 점으로 표시합니다.
  • 🗺 구역 경계 – 담당자별 구역 경계선을 켜고 끌 수 있습니다.
  • 담당자 표시 버튼 – 특정 담당자의 마커와 경계를 개별적으로 보이거나 숨깁니다.
  • 전체 보기 – 모든 마커가 보이도록 지도 범위를 자동 조정합니다.

목록의 지도 → 링크를 누르면 해당 카페를 네이버 지도에서 바로 검색합니다.

💡 알아두면 좋은 사항
  • 방문 상태·보고서·구역 변경은 모두 실시간으로 다른 사용자와 공유됩니다.
  • 보고서 폼은 ✕ 버튼 또는 Esc 키로 닫을 수 있습니다.
  • 지도는 처음 열 때 주소 변환이 진행되며, 이후에는 빠르게 로드됩니다.
